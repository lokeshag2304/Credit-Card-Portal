# 💳 Splendin — Smart Credit Card Portal

Welcome to **Splendin**, a modern and premium fintech credit card management portal designed to provide users with a seamless digital banking experience.

Built with ❤️ using modern web technologies, Splendin empowers users to manage cards, monitor transactions, control payments, explore rewards, and experience next-generation banking — all in one place.

> ✨ *Smart Banking. Elegant Experience. Secure Future.*

# ✨ Features

- 💳 **Interactive Credit Card Dashboard**  
  Manage and monitor all your cards with real-time visuals.

- 📊 **Transaction Analytics**  
  View spending insights, trends, and payment history.

- 🔐 **Secure Card Controls**  
  Freeze/unfreeze cards, manage limits, and security settings.

- 💰 **Rewards & Cashback Tracking**  
  Explore rewards, cashback offers, and loyalty benefits.

- 📱 **Responsive Fintech UI**  
  Smooth experience across desktop, tablet, and mobile devices.

- 🌙 **Dark / Light Mode Support**  
  Modern adaptive theme system with premium UI feel.

- 🔔 **Real-Time Notifications**  
  Track payment alerts, due reminders, and updates instantly.

- 📄 **Statements & EMI Management**  
  Access statements and monitor EMI/loan information easily.

Traditional banking portals often feel outdated, cluttered, and difficult to navigate.

- elegant UI/UX

# 📥 Run Locally

```bash
# Clone Repository
git clone https://github.com/lokeshag2304/Credit-Card-Portal.git

# Enter Project Folder
cd Credit-Card-Portal

# Install Dependencies
npm install

# Start Development Server
npm run dev
````

🌐 Open:

```bash
http://localhost:3000
```
